# North Star — WeMembers

> 最后更新：2026-07-18 · 大股东 I0 已拍板

## 一句话

**新加坡 F&B / 零售门店：零月费上线 → 发代金券与买券抽奖 → 到店核销；平台靠服务费（Model A）赚钱，不靠卖 Token。**

## 产品支柱（已定）

| 支柱 | 对商家的价值 | 对顾客的价值 |
|------|--------------|--------------|
| 代金券 | 获客、跨店结算 | 领券/买券、到店用 |
| 会员 | 复购、等级 | 积分与权益 |
| 幸运抽奖 V2 | 用大奖拉动购券 | 买券即抽、余额再抽 |

## 客群（I0 已确认：P0）

| 优先级 | 客群 | 为什么先做 |
|--------|------|------------|
| P0 | 新加坡 1–10 店 F&B / 轻零售 | 核销频次高、门店二维码场景清晰 |
| P1 | 多门店连锁总部 | 企业账号 + 门店模型已具备 |
| 暂缓 | 纯线上 / 无到店核销 | 与核心路径不符 |

## 商业模型（已定方向）

- **商家侧**：零月费起步；收入来自活动/核销相关服务费（Model A），**不再依赖运营 Token 充值消耗**
- **货币展示**：始终 **S$**，金额以分为单位存储
- **平台账户**：`PLATFORM_ACCOUNT_EMAIL` 控制的平台活动可进 marketplace

## 明确不做（当前）

- 再引入第二套鉴权/支付体系
- 运营 Token 充值卖点
- 未经验证的「全城大奖」市场投放（先单店/小连锁试点）
